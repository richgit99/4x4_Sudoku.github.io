Rules of the game

The Game is similar to the famous 9 x 9 suduko. The only difference that arises in this game is that there will only fours numebrs to be entered instead of nine numebrs and those four numbers are 1,2,3,4.


Each 4 block must contain all those four numbers including the vertcal and horizontal side with no entries being repeated. Moreover the game should be finished within the given time limit.
